# Explore Ceylon – Frontend

A modern Sri Lankan travel management platform built with React + TypeScript + Vite + Tailwind CSS.

## Tech Stack
- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4 (via @tailwindcss/vite)
- React Router DOM 7
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Runs at: http://localhost:5173

## Scripts
- `npm run dev` – Start dev server
- `npm run build` – Production build
- `npm run preview` – Preview production build
- `npm run lint` – Run ESLint
